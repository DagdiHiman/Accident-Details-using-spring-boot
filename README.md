# Accident Details Project
> Made using spring boot
## Functional Requirements
1. Spring Tool Suite or Eclipse IDE 
2. The latest version of the Java Development Kit
3. SOAPUI 
4. Cloud Foundry 
5. XAMPP
6. CanvasJS
